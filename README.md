# VectorSearchDotnet 🧠

Exemplo de integração entre .NET 8 + Clean Architecture + bases de dados vetoriais (Qdrant) para realizar buscas semânticas com embeddings.

🔎 Indexação e consulta de textos com OpenAI + Qdrant;

## 🔧 Funcionalidades

- Geração de embeddings com OpenAI
- Armazenamento em Qdrant
- Consulta por similaridade vetorial
- API REST simples com Clean Architecture
